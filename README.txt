VNDS TRAINING DEMO
==================

Cách chạy:
1. Giải nén thư mục.
2. Mở bằng Live Server trong VS Code.
3. Truy cập index.html.

Cấu hình API Consent:
- Mở Cài đặt Demo > API Consent.
- Bật chức năng gửi dữ liệu về DataTrust.
- Nhập URL API riêng cho từng sản phẩm.
- Nhấn Lưu cấu hình API.

Tên sản phẩm và serviceId được cố định trong js/data.js.
URL API của từng sản phẩm được lưu trong localStorage của trình duyệt.

Lưu ý:
API đích phải cho phép CORS từ địa chỉ chạy website demo.
